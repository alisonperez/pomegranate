Getting Started:

Setup your google form

  1. Make your own copy of the Sample SMS Data Form. (Click on 'file', 'make a copy').
  2. Click View live form, once the page opens copy the URL
  3. Paste the URL here, and click:

Setup your phone

  1. Find a new port by clicking here:
  2. Install SSH Autotunnel
  3. Setup ssh autotunnel. Ask Mike for password and details.
  4. Install SMS Gateway Setup SMS Gateway: Click settings. Check "Listen for HTTP send SMS commands", "Forward incoming SMS to HTTP" and "Enable loggin on SDCARD". Click the HTTP Settings. In the "Listen for HTTP commands on port" , enter the port number that you found above and is now in the form below. In "Forward incoming SMS to HTTP URL", enter "http://sms.ictedge.org/[port number]", for example "http://sms.ictedge.org/2932". Click back until you are on the Status screen. Click Start.

Sending rate. Number of seconds to wait between sending messages. Some phones will stop sending messages if 500 messages in 30 minutes is exceeded (about one every 4 seconds).
